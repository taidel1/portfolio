AI-Powered Financial Assistant for Young Professionals

A hybrid system that combines a multi-sheet financial tracker with AI-powered coaching to help young professionals budget, save, and start investing confidently.

Project Overview

This project is a complete AI-powered financial assistant designed for early-career professionals. It blends structured financial tracking (via Excel/Google Sheets) with a curated AI prompt library that turns any AI model (e.g., Copilot, ChatGPT) into a personal finance coach.

AI & Prompt Engineering Data Modeling Product Design Personal Finance

The Problem

Young professionals often struggle with:

  • Understanding where their money goes each month
  • Building and sticking to a realistic budget
  • Reducing unnecessary spending (subscriptions, impulse buys)
  • Knowing how much they can safely invest
  • Feeling overwhelmed by investing concepts and jargon

Existing spreadsheets and apps are often static and lack personalized guidance or coaching.

The Solution

The AI-Powered Financial Assistant is a three-part system:

  • Finance Tracker Workbook: Multi-sheet Excel/Sheets file with Income, Expenses, Budget, Goals, Investments, and Dashboard views.
  • AI Prompt Library: A collection of structured prompts that turn an AI model into a financial coach.
  • Guide & Case Study: Documentation that explains how to set up, use, and adapt the system.

Key Features

  • Automated KPIs: total income, total expenses, net savings, savings rate
  • Budget vs Actual tracking per category
  • Goal tracking for emergency funds, debt payoff, and investments
  • AI-powered monthly reviews and budget recommendations
  • Long-term investing projections and educational explanations
  • Weekly accountability prompts for habit-building

My Role & Responsibilities

  • Defined the problem and user persona (young professionals 22–35)
  • Designed the multi-sheet financial data model
  • Implemented the tracker with formulas and dashboard metrics
  • Engineered the AI prompt library for reviews, planning, and education
  • Created documentation and a case study for portfolio and product use

Process

1. Research & Requirements

Identified common financial struggles of young professionals and requirements for a system that is both easy to start and sustainable long-term.

2. System Design

Designed the workbook structure (Income, Expenses, Budget, Goals, Investments, Dashboard) and defined KPIs for savings and investing capacity.

3. AI Layer

Created reusable prompts for monthly reviews, budget optimization, savings plans, and investing education, focusing on clear and encouraging language.

4. Documentation

Built a guide that walks users through setup and ongoing routines, plus this case study to explain the project to hiring managers or clients.

Outcomes & Impact

  • Users gain clear visibility into spending and savings behavior.
  • Budget recommendations adapt based on real data and goals.
  • Young professionals can start investing with a simple, educational approach.
  • The project demonstrates practical skills in AI, data, and product design.
